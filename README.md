### [CH563简介](https://github.com/sochub/CH563/wiki)

CH563 SoC是一款32位100MHz ARM9内核MCU，主要用于通信，内置100M以太网MAC和收发器和高速USB设备（全功能USB 2.0 ECHI主机），4组定时器、双异步串口、双SPI接口、ADC、唯一ID等。
