### [CH563简介](https://github.com/sochub/CH563/wiki)

CH563 SoC是一款32位100MHz高速单片机MCU，内置百兆以太网MAC和收发器，内置高速USB设备和全功能USB 2.0 ECHI主机，4组定时器、双异步串口、双SPI接口、ADC、唯一ID等，集成PHY，提供TCP/IP库。
